# WorkBuddy 免费使用腾讯云 Linux 云服务器

1. 下载并安装 WorkBuddy  
https://www.codebuddy.cn/work/

2. 新建一个**项目**（不要直接在聊天窗口运行）。

3. 输入提示词：

```text
帮我部署 https://cnb.cool/aican.do/workbuddy-devtop
workbuddy-devtop 是公开仓库，直接用 HTTPS 公开克隆，无需 Token
```

4. 等待部署完成，打开返回的链接，使用提示的 **admin** 用户名和密码登录。

5. 会话关闭后，点击 **Restart** 即可恢复，数据不会丢失。
